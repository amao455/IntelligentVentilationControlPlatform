# 智能通风管控平台界面布局结构

## 整体架构

```
home-cockpit (根容器)
├── home-background-layer (3D背景层)
│   └── LazyHomeObjBackground (3D巷道模型)
│
└── home-content-layer (内容层)
    ├── home-kpi-row (KPI指标行)
    │   └── 8个KPI卡片 (响应式布局)
    │
    └── home-main-layout-row (主内容区 - 三栏布局)
        ├── 左栏 (18.75%宽度)
        ├── 中栏 (自适应宽度 - 显示3D模型)
        └── 右栏 (18.75%宽度)
```

---

## 一、顶层容器 (home-cockpit)

- **作用**: 根容器
- **布局**: 相对定位，包含背景层和内容层
- **尺寸**: 100vw × 100vh
- **溢出**: hidden

---

## 二、背景层 (home-background-layer)

- **内容**: 3D巷道模型 (LazyHomeObjBackground)
- **特性**: 
  - 懒加载 (延迟350ms)
  - 固定位置
  - 透明度: 0.55
  - 禁用交互

---

## 三、KPI指标行 (home-kpi-row)

### 响应式列数
- xs (< 576px): 1列
- sm (≥ 576px): 2列
- md (≥ 768px): 3列
- lg (≥ 992px): 4列
- xl (≥ 1200px): 8列

### 内容
- 8个KPI卡片，显示关键指标

---

## 四、主内容区 - 三栏布局

### 左栏 (18.75%宽度)

```
主通风机运行状态 (固定高度350px)
├── Tabs: 1号机 / 2号机
└── 每台机器显示:
    ├── 运行状态指示
    ├── 6项实时参数
    │   ├── 电流: 245A
    │   ├── 电压: 6000V
    │   ├── 功率: 1200kW
    │   ├── 转速: 740rpm
    │   ├── 风量: 8500m³/min
    │   └── 负压: -2800Pa
    └── 运行时长进度条
```

---

### 中栏 (自适应宽度)

- **作用**: 透过此区域显示3D巷道模型
- **特性**: 留空，禁用交互

---

### 右栏 (18.75%宽度) - Flexbox垂直布局

```
home-right-column-stack (Flexbox容器)
├── 1. 设备运行状态
│   ├── 布局: flex: 0 0 auto, min-height: 180px
│   └── 内容:
│       ├── 主扇运行: 2台
│       ├── 局扇运行: 14套
│       └── 风窗调节: 3套
│
├── 2. 24h总风量趋势
│   ├── 布局: flex: 0 0 auto, min-height: 160px
│   └── 内容: ECharts折线图 (130px)
│
└── 3. 预警报警
    ├── 布局: flex: 1 1 0, min-height: 200px
    ├── 统计摘要: 严重/告警/预警
    └── 报警列表: 最多8条，可滚动
```

---

## 五、右栏布局逻辑 (核心)

### Flexbox配置

```css
.home-right-column-stack {
  display: flex;
  flex-direction: column;
  gap: 10px;
  height: 100%;
}

/* 设备运行状态 */
.home-right-column-stack > *:nth-child(1) {
  flex: 0 0 auto;
  min-height: 180px;
}

/* 24h总风量趋势 */
.home-right-column-stack > *:nth-child(2) {
  flex: 0 0 auto;
  min-height: 160px;
}

/* 预警报警 - 占据剩余空间 */
.home-right-column-stack > *:nth-child(3) {
  flex: 1 1 0;
  min-height: 200px;
}
```

### 布局原理

1. **前两个卡片**: `flex: 0 0 auto` - 不伸缩，根据内容和最小高度
2. **第三个卡片**: `flex: 1 1 0` - 占据所有剩余空间
3. **gap: 10px** - 卡片间距
4. **min-height** - 确保每个卡片有最小显示高度

## 六、响应式断点

### 大屏 (≥ 1200px)
- 布局: 三栏并排
- KPI: 8列

### 中屏 (992px - 1199px)

- KPI: 4列
- 右栏卡片: 160px / 140px / 180px

### 小屏 (< 992px)
- 布局: 单列堆叠
- KPI: 2-3列
- 右栏卡片: 140px / 120px / 160px

---

## 总结

智能通风管控平台采用**三栏布局**设计：

- **左栏**: 主通风机运行状态（固定350px）
- **中栏**: 3D巷道模型展示区（自适应）
- **右栏**: 设备状态 + 趋势图 + 报警（Flexbox垂直堆叠）

**右栏核心特点**：
- 前两个卡片固定最小高度
- 第三个卡片使用 `flex: 1` 占据剩余空间
- 确保界面充分利用纵轴空间，无溢出

**整体风格**：科技感工业监控界面，深色背景、青色主题、透明卡片、扫描线动画。
